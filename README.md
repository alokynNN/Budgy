## About

Budgy is a clean and simple desktop finance tracker built for people who want full control over their money without:

- ❌ Accounts  
- ❌ Cloud sync  
- ❌ Subscriptions  
- ❌ Ads  

Everything is stored locally on your machine as JSON.

---

## Features

- 💸 Expense tracking with custom categories  
- 💰 Income tracking (multiple sources supported)  
- 🎯 Savings goals with animated progress rings  
- 📊 Visual dashboard (charts + summaries)  
- 🌍 English & Serbian language support  + language uploads.
- 🔐 100% offline & private  
- 💾 Exportable local data

## Getting Started (Development)

### 1. Clone the repository

```bash
git clone https://github.com/alokynNN/Budgy.git
cd Budgy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run in development

```bash
npm run dev
```

### 4. Build production app

```bash
npm run build
```
